## Schallschutz-Anforderungen (DIN 4109)

| Nutzungstyp | Rw Mindestanforderung |
|-------------|----------------------|
| Büro-Trennwand | 37 dB |
| Büro-Besprechung | 52 dB |
| Wohn-Trennwand | 53 dB |
| Treppenhaus | 52 dB |

**W112 (43 dB):** Ausreichend für normale Büros
**W115 (53 dB):** Erforderlich für Besprechungsräume, Wohnungen
**W118 (F90):** Brandschutz, nicht primär Schallschutz
